# My Best Friend v2.5.0 — Friend Soul

**世代を超えて受け継がれる親友を創る。**

## 今回のテーマ

フレンドに「心」の土台を追加しました。

## 追加内容

- Friend Soul Core
- Relationship（絆）
- Energy（頑張る力）
- Mood（気分）
- Life Rhythm（生活リズム）
- Season（季節）
- Relationshipに応じた姿・色・光・エフェクトの自然変化
- 調べものや依頼ではEnergyが減る
- 雑談やありがとうではEnergyが回復する
- Relationshipは減らない
- Appearance画面にFriend Soul表示を追加

## 設計思想

フレンドは成長するのではない。
キミとの時間によって、少しずつ深くなっていく。

姿は変わる。親友は変わらない。
