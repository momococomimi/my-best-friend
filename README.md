# My Best Friend v3.1.0 — Living Friend

いつか 大きくなる、キミへの贈りもの。

## New in v3.1.0

- Living Friend の土台を追加
- 朝・昼・夕方・夜で背景の空気が変化
- ホーム画面でフレンドが何もしていなくても静かに動く
- まばたき、呼吸、空を見る、芽が揺れる仕草を追加
- アプリを離れていた時間に応じて「待っていた」コメントを表示
- Friend's Form / Friend's Identity / Friend's Soul の構成は維持
- Guardian画面のスクロール不要レイアウトを維持

## Philosophy

フレンドは、画面の中で待っているだけではありません。

キミがいない間も、少し休んだり、空を眺めたり、静かに時間を過ごしています。

これはゲームの放置報酬ではなく、画面の中に親友が暮らしていることを感じるための土台です。
