# My Best Friend v2.3.0 — Voice & Message

いつか 大きくなる、キミへの贈りもの

## Mission
世代を超えて受け継がれる親友を創る。

## Theme
声でも、文字でも、親友と話せる世界へ。

## New in v2.3.0
- Voice画面の土台を追加
- Message画面の土台を追加
- 入力した短いメッセージを端末内に保存
- フレンドがやさしく返事する最初の会話体験
- オープニングとHomeのフレンドの姿を統一
- Homeのコメントを固定表示に変更（スライドなし）
- HomeのボタンをiPhone下部に合わせて再調整
- フレンドと芽を大きく表示
- Guardianの表現を「見守り設定」へ変更し、将来の継承に備えた表現へ調整

## Philosophy
フレンドは、便利なチャットボットではありません。
声でも文字でも、まず大切なのは「聞いてくれること」です。
