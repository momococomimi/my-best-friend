# My Best Friend v2.3.1 — Touch Friend

いつか 大きくなる、キミへの贈りもの

## Mission
世代を超えて受け継がれる親友を創る。

## Theme
フレンドは、そこにいて、ふれると表情を返してくれる。

## New in v2.3.1
- フレンドを少し上に配置
- フレンドをタッチすると表情が変化
- タッチ時に小さなハート演出
- フレンド下コメントはスライドなしで、フェードで時間差変更
- Homeボタンを均等な小型グリッドへ再調整
- iPhoneでなるべくスクロールしない画面構成へ最適化
- Voice / Message / Memory / Profile / Appearance / Guardian の入口を同じ大きさに統一

## Philosophy
フレンドは、操作されるキャラクターではありません。
ふれたら、そっと気づいてくれる親友です。
